# Countdown Next Year App

Displays the remaining time until the next year from the current time and plays a special animation at the moment of the New Year's transition.


## Technology Stacks
- TailwindCSS
- Node.js
- React 19
- Vite
- CloudFlare Pages


## Languages
- TypeScript
- JavaScript
- CSS


## Getting Started

### Prerequisites
- [pnpm](https://pnpm.io/installation)

### Installation
  1. Clone the repository
  ```sh
  git clone https://github.com/tukuyomil032/countdown-next-year-app.git

  cd countdown-next-year-app
  ```


  2. Install dependencies
  ```sh
  pnpm install
  ```


  3. Run
  ```sh
  pnpm dev
  ```
